# YouJoin-Android
## 简介
功能和特点包括：
 - 采用Material Design设计
 - 登录注册
 - 即时聊天
 - 个人中心（支持资料编辑、头像上传）
 - 心情动态（支持最多九张图片；支持点赞、评论）
 - 好友关注
 - 附近的人（采用**特征点匿名匹配算法**，服务器不了解客户端真实地理位置，保护用户隐私）
 - 支持emoji表情（原创emoji表情绘制中）
 - 插件扩展（开发中）
